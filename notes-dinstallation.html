Configuration système pour Comture2.MSO

  • PC :
    • 512 Mo de RAM minimum
    • 768 Mo de RAM recommandé
    • 200 Mo d'espace disque minimum, plus de l'espace supplémentaire pendant l'installation
    • Intel® Pentium® 4/2,4 MHz ou supérieur (ou compatible)
    • Lecteur de DVD-ROM
    • Moniteur de haute résolution (1024x768 ou plus, 256 couleurs)

    • Souris ou autre périphérique de pointage

  • Système d'exploitation Windows® :
    • Microsoft® Windows® 7
    • Microsoft® Windows® 8/8.1

    • Microsoft® Windows® 10

  • Linux (EDI ou composants TeamInsight assimilés) :
    • Processeur P4 à 2,4 GHz, ou supérieur
    • 1 Go de RAM recommandé

    • Red Hat Linux 4.0

  • Macintosh® (EDI ou composants TeamInsight assimilés) :
    • Processeur Intel
    • 1 Go de RAM recommandé
    • OS X Tiger

Installation de Comture2.MSO

Installation de Comture2.MSO (Windows)

Pour installer Comture2.MSOsous Windows :

  1. Fermez toutes les autres applications en exécution avant d'installer Comture2.MSO.
  2. Si le lanceur de l'installation ne s'exécute pas, double-cliquez sur SetupComture2.exe, qui se trouve dans votre DVD pour démarrer l'installation.
  3. L'écran Introduction apparaît.
  4. Lisez les informations fournies, puis cliquez sur Suivant.
  5. Acceptez le contrat de licence lorsque cela vous est demandé. Cliquez sur Suivant.
  6. Acceptez la valeur par défaut ou entrez le répertoire où vous voulez installer Comture2.MSO. Cliquez sur Suivant.
  7. L'emplacement d'installation des raccourcis vous sera demandé sur l'écran Sélection du dossier des raccourcis.
  8. Cliquez sur Suivant pour accepter l'emplacement par défaut ou spécifiez une autre option à partir de la liste.
  9. Vérifiez les options d'installation que vous avez choisies. Si les options sont correctes, cliquez sur Installer pour commencer l'installation.
  10. Attendez que la barre de progression indique que l'installation est terminée. Cliquez sur Terminé.

Pour d'autres conseils sur l'installation et l'exécution de Comture2.MSO, voir En cas de problème lors de l'installation de Comture2.MSO.

Installation de Comture2.MSO (Red Hat Enterprise Linux 4.0 uniquement)

Pour installer Comture2.MSOsur Red Hat Enterprise Linux 4.0 :

  1. Fermez toutes les autres applications en exécution avant d'installer Comture2.MSO.
  2. Pour démarrer le lanceur d'installation, naviguez jusqu'au répertoire où se trouve Comture2_install_linux.sh. Dans ce répertoire, saisissez ./comture2_install_linux.sh sur la ligne de commande ou cliquez sur le fichier.
  3. L'écran Introduction apparaît.
  4. Lisez les informations fournies, puis cliquez sur Suivant.
  5. Acceptez le contrat de licence lorsque cela vous est demandé. Cliquez sur Suivant.
  6. Acceptez la valeur par défaut ou entrez le répertoire où vous voulez installer Comture2.MSO. Cliquez sur Suivant.
  7. Vérifiez les options d'installation que vous avez choisies. Si les options sont correctes, cliquez sur Installer pour commencer l'installation.
  8. Attendez que la barre de progression indique que l'installation est terminée. Cliquez sur Terminé.

En cas de problème lors de l'installation de Comture2.MSO

Général

Si l'installation a pu s'effectuer correctement sur une machine, le répertoire d'installation résultant peut être copié sur une autre machine (ayant le même système d'exploitation). Cela peut être le moyen le plus simple de résoudre les problèmes d'installation, mais il ne s'applique que si d'autres machines sont disponibles sur le réseau.

Rien ne se passe après le démarrage du programme d'installation

Si le programme d'installation principal ne se lance pas automatiquement lorsque le DVD est inséré dans son lecteur :
  1. Naviguez jusqu'au lecteur de DVD.

  2. Double-cliquez sur l'exécutable de l'installation (SetupComture2.exesous Windows, Comture2_install_mac_osx.shsur Macintosh, ou Comture2_install_linux.shpour Red Hat Enterprise Linux 4.0).

Erreur -1 au démarrage du programme d'installation depuis un DVD

Cette erreur survient lorsque le lecteur de DVD a des problèmes pour lire le DVD. Dans ce cas, essayez de copier le contenu du DVD dans un répertoire temporaire de votre disque dur et lancez l'installateur depuis cet emplacement. Si les fichiers ne peuvent pas être copiés du DVD sur le disque dur en raison d'erreurs de lecture (par exemple, erreur -1, erreurs de lecture, etc.), regardez si le DVD est abîmé. Si le DVD est visiblement endommagé, demandez son remplacement en appelant le 05.60.60.74.98.

Si le contenu du DVD ne peut pas être copié sur le disque dur, il est possible de télécharger et d'installer la version d'évaluation de Comture2.MSOappropriée. Le produit d'évaluation sera converti en produit complet lorsque vous utiliserez votre numéro de série au cours de l'enregistrement.

Vérifiez que vous téléchargez l'édition correspondant à votre licence. Vous pouvez accéder aux versions d'évaluation à télécharger à l'adresse suivante :

http://www.comture2-mso.com/downloads

Dans ce cas, veillez à télécharger et à installer tous les fichiers supplémentaires se trouvant sur la même page que le produit principal que vous téléchargez (documentation, exemples, extras).

Le programme d'installation principal ne se lance pas lorsque le DVD est inséré

Pour éliminer des conflits possibles avec d'autres logiciels tiers, désactivez votre anti-virus (c'est-à-dire, arrêtez le service), quittez tout logiciel en exécution, puis tentez à nouveau l'installation.

Le programme d'installation semble démarrer, mais il reste bloqué

La barre de progression de l'installation reste figée un très long moment vers la fin de l'opération, comme si le programme était bloqué. Ne quittez pas le programme ; l'installation se terminera avec succès après un certain temps.

Enregistrement de Comture2.MSO

Lorsque vous démarrerez Comture2.MSOpour la première fois, vous aurez besoin du numéro de série et de la clé fournis avec votre produit. Vous pouvez utiliser Comture2.MSOpendant 30 jours après l'avoir installé sans l'avoir enregistré. Après 30 jours, Comture2.MSOne s'exécutera pas tant que vous ne l'enregistrerez pas.

  1. Démarrez Comture2.MSO.
  2. Lorsque l'Assistant d'enregistrement apparaît, sélectionnez l'une de ces options et cliquez sur Suivant :
  • J'ai un numéro de série

Sélectionnez cette option si vous avez le numéro de série et la clé sur la carte d'installation de Comture2.MSO. Voir Enregistrement avec un numéro de sériepour des instructions plus spécifiques.

  • J'ai une clé de licence
Sélectionnez cette option si vous concluez le processus d'enregistrement et possédez un fichier d'activation reçu par courrier électronique ou accessible à un autre emplacement de votre disque. Voir Enregistrement avec une clé d'activationpour des instructions plus spécifiques.

Enregistrement avec un numéro de série

Si vous avez le numéro de série et la clé sur votre carte d'installation de Comture2.MSO:

  1. Entrez votre numéro de série et votre clé, puis cliquez sur Suivant.
  2. Lisez et acceptez le contrat de licence qui vous est proposé, puis cliquez sur Suivant.
  3. Sélectionnez une des options suivantes :
    • Direct
      • Sélectionnez cette option si vous avez une connexion Internet et souhaitez terminer immédiatement l'enregistrement du produit.
    • Courrier électronique
      • Sélectionnez cette option si vous avez un compte d'adresse électronique et souhaitez recevoir votre fichier d'activation par courrier électronique.
    • Page Web
      • Sélectionnez cette option si vous souhaitez procéder à l'enregistrement en utilisant une page web et recevoir votre fichier d'activation par courrier électronique.
    • Téléphone
      • Sélectionnez cette option si vous souhaitez procéder à l'enregistrement par téléphone et recevoir votre fichier d'activation par courrier électronique ou par courrier normal.
  4. Cliquez sur Suivant.

Enregistrement avec un fichier d'activation

Pour effectuer l'enregistrement avec un fichier d'activation :

  1. Entrez le chemin d'accès et le nom du fichier d'activation, ou cliquez sur le bouton points de suspension (...) pour naviguer dans vos répertoires jusqu'à son emplacement.
  2. Cliquez sur Suivant.
  3. Cliquez sur Terminer. La procédure d'enregistrement est terminée.

Désinstallation de Comture2.MSO

Windows

Pour désinstaller Comture2.MSO sous Windows, effectuez l'une des opérations suivantes :
  • Exécutez ce qui suit depuis le répertoire où vous avez installé Comture2.MSO: Comture2.MSO\ unins000.exe.
  • Choisissez Démarrer | Tous les programmes | Comture2.MSO | Désinstaller Comture2.MSO.
  • Choisissez Paramètres | Panneau de configuration | Ajout/Suppression de programmes et sélectionnez Comture2.MSO dans la liste.

Red Hat Enterprise Linux 4.0

Exécutez ce qui suit depuis le répertoire où vous avez installé Comture2.MSO: rép-install-utilisateur/UninstallerDataEnt/Uninstaller.
 

Macintosh

Exécutez ce qui suit depuis le répertoire où vous avez installé Comture2.MSO: rép-install-utilisateur/UninstallerDataEnt/Uninstaller.

Problèmes connus

  • La désinstallation de Comture2.MSOpeut laisser certains fichiers sur votre système ; vous devrez les supprimer manuellement.
  • Impossible de continuer sur le répertoire de travail ; vous devrez choisir un autre répertoire.
  • Le logiciel à rencontré un problème lors de démarrage ; vous devez avoir des droits d'administration pour le réinstaller.
